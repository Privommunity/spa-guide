# 給湯設備故障時の温泉利用補償案内

管理物件の給湯設備故障に伴い、お風呂が使用できない期間における温泉利用の補償についてご案内するランディングページです。

## 公開URL

GitHub Pagesで公開中：https://privommunity.github.io/spa-guide/

## 補償内容

- 大人1人当たり日額最大1,000円まで補償
- 契約者及び申請されている同居人分のみが対象
- 領収書提示による後日清算

## 使用方法

入居者にSMSでURLを送信し、補償内容を確認していただけます。
